# Pfad Parameter
Pfad Parameter sind Daten die in dem URL Pfad stecken z.B. `example.com/greet/max`.
Hierbei ist der teil `max` Variable und wird ausgelesen.

```py
from fastapi import FastAPI

app = FastAPI()

@app.get("/greet/{name}")
async def greet(name: str):
    return f"Hello {name}!"
```
